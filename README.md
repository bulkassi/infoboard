# Infoboard (Vue + FastAPI + PostgreSQL)

This project uses:
- Frontend: Vue 3 + Vite (JavaScript), Vue Router, Pinia, ESLint, Prettier
- Backend: FastAPI + SQLModel + Alembic
- Database: PostgreSQL
- Orchestration: Docker Compose

## Quick start
1. Copy `.env.example` to `.env`.
2. Scaffold frontend (already initialized) and install dependencies.
3. Create backend virtual environment and install backend dependencies.
4. Run `docker compose up --build`.
